## Jasmin UI

Demo http://fe.jimu.io/jasmin-ui

#### Usage

```bash
yarn add @jimengio/jasmin-ui
```

Or

```bash
npm install @jimengio/jasmin-ui
```

注：项目入口文件要引入 css 文件

```ts
import "@jimengio/jasmin-ui/lib/assets/antd/antd.min.css"
```

安装开发依赖

```ts
yarn install
```

启动项目

```bash
yarn dev
```

编译

```bash
yarn release
```
